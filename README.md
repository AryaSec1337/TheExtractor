# TheExtractor

A simple and interactive Python-based tool to extract endpoints, URLs, and JavaScript file links from text logs/files.

## Features

- **Extract Endpoints**: Extracts all relative paths, absolute paths, and full URLs.
- **Extract JS**: Extracts only JavaScript file URLs/paths (ending with `.js`).
- **Extract URLs**: Extracts only complete HTTP/HTTPS links.

## Usage

Run the script interactively:
```bash
python TheExtractor.py
```
And follow the interactive prompts!
